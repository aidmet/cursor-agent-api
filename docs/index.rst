cursor-agent-sdk
================

Python client for the `Cursor Cloud Agents API <https://cursor.com/docs/cloud-agent/api/endpoints>`__
— start cloud coding agents on GitHub repos or PRs, send follow-ups, and poll status. Sync and
async :mod:`httpx` clients are both supported.

Install from PyPI: ``pip install cursor-agent-sdk`` (`project page <https://pypi.org/project/cursor-agent-sdk/>`__).
